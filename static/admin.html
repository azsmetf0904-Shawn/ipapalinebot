運作方式說明
📨 使用者傳訊息給機器人後，系統依序執行：
1. 比對「課程/會議/招商/培訓」關鍵字 → 命中 → 直接查 DB 回覆（不過 AI）
2. 比對「公告」關鍵字 → 命中 → 直接查 DB 回覆（不過 AI）
3. 都沒命中 → 進 Gemini AI，AI 拿到完整資料庫背景 + 意圖標籤 → 回覆
意圖標籤讓 AI 知道「這題要查課程」還是「這題是閒聊」，避免 AI 引用不該引用的資料或漏列資料。
